# How to Start
- fork this repository
- clone
```zsh
git clone https://github.com/veldanava/HacktoberFest2023.git
```
- create new branch
```zsh
git checkout -b kiana (example)
```
```zsh
git add .
```
- commit your branch
```zsh
git commit -m "Lorem Ipsum!!!"
```
- and push
```zsh
git push origin (your branch)
```
